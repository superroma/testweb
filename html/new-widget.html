Adding new widget

Choose widget type

Chart

Chart is handy for showing data behavior over time.

Gauge

A gauge is a read only block type that shows a fixed range of values.

Switch

A toggle button is useful if you have an ON or OFF type of state. You can configure what values are sent.

Slider

The slider works well if you have a range of values you need to send.

String

A string block can be used to send data as well as view data.

Number

A number block displays a plain number - large and fine.
